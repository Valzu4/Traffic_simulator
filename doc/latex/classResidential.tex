\doxysubsection{Detailed Description}
\mbox{\hyperlink{classResidential}{Residential}} building represents an residential building in the traffic simulation. 

\mbox{\hyperlink{classResidential}{Residential}} building is one type of building where the residents are implemented. The capacity is set to 100 by the program but user can also change it.
