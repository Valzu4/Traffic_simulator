\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSquare}{Square}} class is a building block of city, and might contain buildings or roads. 

\mbox{\hyperlink{classSquare}{Square}} is a building block of the city. It can contain buildings, roads or be empty. Empty squares simulate empty spaces in the city, and don\textquotesingle{}t interract with other squares.
