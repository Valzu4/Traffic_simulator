\doxysubsection{Detailed Description}
\mbox{\hyperlink{classIndustrial}{Industrial}} building represents an industrial building in the traffic simulation. 

\mbox{\hyperlink{classIndustrial}{Industrial}} building is one type of building for the residents who are going to work. The capacity is randomized by the program so the user can\textquotesingle{}t affect it.
