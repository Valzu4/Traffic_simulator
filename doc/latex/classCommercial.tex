\doxysubsection{Detailed Description}
\mbox{\hyperlink{classCommercial}{Commercial}} building represents an commercial building in the traffic simulation. 

\mbox{\hyperlink{classCommercial}{Commercial}} building is one type of building for the residents who are going to stores, gyms, restaurants etc. The capacity is randomized by the program so the user can\textquotesingle{}t affect it.
